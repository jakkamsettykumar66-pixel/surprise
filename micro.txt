>To list the branches present locally: git branch or git branch --list
>To create a branch: git branch branch_name
>To switch to a branch : git switch branch_name or git checkout branch_name
>To create and switch to a branch simultaneously : git checkout -b branch_name
>To push any branch to a remote repo : git push alias_naame branch_name
 git push origin master/main

>To delete the branch locally : git branch -d branch_name / git branch -D branch_>
>To delete the branch remotely: git push master/main -d branch_name
>To rename a branch : git branch -m old_branch_name new_branch_name
